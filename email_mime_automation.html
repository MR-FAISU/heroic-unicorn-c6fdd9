Automation rules

Top-level headers: Subject: {sub}, From: "{from}" <{listid}>, Date: {hdt}, To: {to}. If the HTML part is Base64 or quoted-printable, it is decoded as UTF-8 and changed to Content-Transfer-Encoding: 7bit. If </style> is present, all inner closing style tags are removed and one closing style tag is placed at the end. If it is not present in a full HTML document, the body is kept visible. Non-document fragments without a closing style tag are hidden inside a comment tag. If a MIME boundary exists, the final closing boundary is removed.
